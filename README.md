Here is the digital design implementation of all basic to advanced designs using verilog HDL for understanding purpose only.

comprehensive list of all designs are as follows : 

1. Adders -> Half Adder,
             Full Addeer,
             BCD Adder,
             Look Ahead Carry Adder,
             Ripple Carry Adder.

2. Multiplier -> Booth Multiplier,
                 N bit Multiplier,
                 Vedic Multiplier.

3. Arithematic and Logical Unit - > 4 Bit ALU,
                                    16 Bit ALU.

4. Basics Designs -> Decoder (3 to 8),
                     Encoder (8 to 3),
                     Mux  (3 Select Lines),
                     Demux (3 Select Lines),
                     Shift Register,
                     FIFO.
   
6. Flip Flops -> D Flip Flop,
                 T Flip Flop,
                 SR Flip Flop,
                 JK Flip Flop.

7. Memory Unit -> ROM,
                  Single Port RAM,
                  Dual Port RAM.

8. FSM based -> Vending Machine,
                Voting Machine.
